Comment interpréter ces colonnes ?

Taux prédit = taux d'exécution attendu (Engage ÷ Crédits) calculé par la méthode optimale pour chaque ligne, choisie par backtest sur 2024-2025.
Méthode : RateLast = dernier taux observé · RateMean = moyenne historique · RateMed = médiane (robuste) · RateTrend = tendance linéaire.
Engage prédit = Crédits scénario × Taux prédit.

Données 100% locales. Aucune requête réseau effectuée — confidentialité garantie.
